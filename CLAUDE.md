# JSY Food 재고수불부 생성기

## 프로젝트 목적
JSY Food F&B의 재고수불부(소스/원육/가공육) 자동 생성 도구

## 작업 원칙
- **정확성 최우선**: 빠르고 적당히보다 느리고 완벽히
- 엑셀 데이터 검토 시 누락 없이 철저하게
- 수식 기반 엑셀 출력 선호 (하드코딩 금지)
- 청기와 5색 테마: 남색(#122437), 녹색(#006E32), 적갈색(#CB4427), 금색(#E2BC54), 청색(#143C73)
- 테이블 배경 흰색 + 폰트 컬러로 구분 (기초=청색, 입고=녹색, 출고=적갈색, 기말=남색)

## 검증 레이어 (핵심)
정확성 최우선 → 데이터가 올바르게 반영되었는지 사용자가 직접 확인할 수 있어야 함

### 1. 파싱 미리보기
- 작업일지 업로드 시 바로 수불부에 반영하지 않고, 중간 테이블로 파싱 결과 보여줌
- 품목명 매핑 결과 (원본명 → 수불부 품목명)
- 월합계 수량, 원본값 vs 파싱값 나란히 표시

### 2. 교차 검증 대시보드
- 입고: 품목별입고현황 수량 vs 작업일지 입고 수량 → 차이 하이라이트
- 수불 정합성: 기초 + 입고 - 출고 = 기말 → 불일치 시 빨간색
- 전월 연결: 전월 기말 = 당월 기초 → 불일치 시 경고

### 3. 원본 추적 (툴팁)
- 각 셀에 마우스 올리면 출처 표시 (예: "참프레시 작업일지 2601, R7")

### 4. 차이 리포트
- 엑셀 다운로드 전 "검증 결과: 불일치 0건 / 경고 2건" 요약

## 출력 형태
- 단일 HTML 파일 (index.html)
- 엑셀 다운로드: 파일별 개별 다운로드 (소스 / 원육 / 가공육 각각)
- xlsx-js-style 사용

## 참고 엑셀 구조

### 1. 소스 수불부 (재고수불부_소스_25년_총평균.xlsx)
- 시트: 2512, 2511, 원재료매입
- 월별 시트 구조: 품목명 | 단위 | 기초(수량,단가,금액) | 입고(수량,단가,금액) | 출고(수량,단가,금액) | 기말(수량,단가,금액)
- 원재료매입 시트: 월별(1~12월) 구매현황 (수량,단가,금액)
- 총평균법 적용

### 2. 원육 수불부 (재고수불부_원육_25년_총평균.xlsx)
- 시트: 2501~2512 (12개월 전체), 매입통합
- 월별 시트 구조: 전사 기준, 기초/입고/출고/기말 + 입고비고(컨테이너+육가공 우사각 입고) + 출고비고(육가공+우사각 출고)
- 매입통합 시트: 월별 구매현황
- 단위: Kg, 총평균법 적용

### 3. 가공육 수불부 (재고수불부_가공육_25년_총평균.xlsx)
- 시트 (23개): 출고가, 연간, 2501~2512, 원육미지급금현황, 가공육수불부용, 손익, 수율, 공장, 가공비, 본푸드, 선물세트, 공장간이동
- 월별 수불부: 전사 기준, 기초/입고/출고/기말 (수량,단가,금액) + 안창살/우설 비고
- 부가 시트: 출고가현황, 손익분석(월별 판매가-출고단가-가공비=이익), 수율, 공장운영, 가공비, 본푸드 재고, 선물세트 구성
- 가장 복잡한 구조, 총평균법 적용
